# Test automation

---

### What to automate

First thing first, a complete publish flow with photo gallery needs to be automated and integrated with CI. All key user flows are planned to be automated.

### How to automate

For Android UI test automation, the first choice would be [**Espresso**](https://developer.android.com/training/testing/ui-testing/espresso-testing.html\), if the tests are going to be maintained by developers and Espresso [**IdlingResource**]\(https://developer.android.com/reference/android/support/test/espresso/IdlingResource.html) interface has been broadly implemented across the app, so Espresso tests could easily register to it for easy synchronized waiting.
